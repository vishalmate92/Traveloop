# 🌍 Traveloop

An AI-powered travel planning and budget management platform built for hackathons and modern travelers.

Traveloop helps users plan smarter trips with AI-generated itineraries, budget analytics, destination management, and real-time travel organization tools — all in one modern web application.

---

## 🚀 Features

* ✨ AI-powered travel planning
* 📍 Smart itinerary generation
* 💰 Budget analytics and expense tracking
* 📊 Interactive travel dashboard
* 🔐 Firebase integration and authentication support
* 📱 Responsive modern UI
* ⚡ Fast performance with Vite + React
* 🎨 Beautiful user experience using Tailwind CSS and shadcn/ui

---

## 🛠️ Tech Stack

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/traveloop.git
cd traveloop
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env.local` file in the root directory.

```env
GEMINI_API_KEY=your_gemini_api_key
```

---

## ▶️ Run Locally

Start the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Run production build:

```bash
npm start
```
